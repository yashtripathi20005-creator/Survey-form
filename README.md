# Survey Form Application

A complete, responsive survey form built with Flask (Python), HTML, CSS, and JavaScript.

## Features

- 🎨 Modern, responsive design with gradient backgrounds
- ✅ Client-side and server-side validation
- 📊 Stores survey responses in JSON format
- 🔄 AJAX form submission with real-time feedback
- 📱 Mobile-friendly interface
- 🎯 Interactive form elements with hover/focus effects

## Tech Stack

- **Backend**: Flask (Python 3.7+)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Data Storage**: JSON file
- **Styling**: Custom CSS with animations
